CellometerAI

Automated cell counting in microscope images

Automating Cell Counting

https://github.com/JMtSG/CellometerAI

This project was created to assist with my friend's PhD work. The task at hand was to count the number of protoplasts in each well, to use for normalisation of measurements. Fortunately, photographs of each well could be captured through a microscope in her lab, which led to the idea of developing a tool to automatically count the number of cells, rather than relying on interns to do so manually.

Prior to this, the extent of automated tools available to us for this task used classical image processing techniques such as Watershed segmentation, which required significant image pre-processing and was not robust against varying exposure and scale.
With the advent of open-source CNN-based image object detection models, it was trivial to take an existing pre-trained model such as Ultralytics YOLO and repurpose it for our needs. The model simply requires a small amount of hand labelled data for fine-tune training, and some wrappers written around it to assist with image pre-processing and collation of model output data.

This completed tool takes in a folder of microscope photographs, runs a sliding window over each to generate tiles that are passed through the YOLO model, filters and counts the detections, then provides a spreadsheet summary and annotated output images for review.
